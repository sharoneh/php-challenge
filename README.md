# CrossKnowledge Code Challenge
This web application uses [Parcel](https://parceljs.org/) as a bundler.

```bash
# To install the dependencies
$ yarn

# To run a local server in development mode
$ yarn dev

# To run a local server in production mode
$ yarn build
```

## Features

- **Cache function**: PHP caching function with [Redis](https://redis.io/).

- **Date formatting**: JavaScript code that replaces the innerHTML value (that is initially an ISO date format string) from elements with the class `js-date-format` with the formatted value of the time passed since the element initial time.
